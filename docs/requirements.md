# Требования: Новый Личный Сайт (Фотогалерея + Блог)

## 1) Цель Проекта
- Построить новый персональный сайт, где основной контент это фотографии.
- Текстовый блог остается вторичным каналом для редких записей.
- Сайт публикуется как статический через GitHub Pages.

## 2) Информационная Архитектура
Обязательные страницы:
- `Home`
- `Gallery`
- `Photo`
- `Blog`
- `Post`
- `About`
- `404`

## 3) Контентные Приоритеты
- Приоритет 1: фото, фотосерии, визуальная подача.
- Приоритет 2: короткие и редкие текстовые публикации в блоге.

## 4) Визуальное Направление
- Типографика: выразительная пара шрифтов для заголовков и текста, с хорошей читаемостью на мобильных.
- Палитра: нейтральная база (светлый фон/графитовый текст) с одним акцентным цветом для интерактивных элементов.
- Плотность сетки: плотная, визуально насыщенная фото-сетка без перегрузки текстом.
- Адаптивность: mobile-first, корректная работа на 320px+ и десктопе.

## 5) Критерии Приемки: Галерея
- Есть страница `Gallery` со списком фото/серий.
- Есть страница `Photo` с крупным изображением и метаданными.
- Навигация между фото работает (например, предыдущая/следующая).
- Превью в галерее не блокируют первый экран (lazy-loading/оптимизация).
- Нет битых ссылок на изображения.

## 6) Критерии Приемки: Блог
- Есть страница `Blog` со списком публикаций.
- Есть страница `Post` с корректной типографикой текста.
- Пост открывается по прямой ссылке и из списка.
- Поддерживаются теги постов.
- Черновики не попадают в production-сборку.

## 7) Нефункциональные Требования
- Производительность: целевая оценка Lighthouse mobile Performance >= 90.
- Доступность: целевая оценка Lighthouse mobile Accessibility >= 90.
- SEO: целевая оценка Lighthouse mobile SEO >= 90.
- Деплой: полностью автоматический через GitHub Actions на GitHub Pages.
